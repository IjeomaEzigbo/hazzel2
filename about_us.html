DIFFERENCES BETWEEN HTML ELEMENTS AND TAGS

HTML TAGS

Html tags represent the root of an html document. Html tag is the container for all other html elements except the tag. They are labels used to start and close an element except the empty elements which do not have closing tags. Tags are enclosed in angled brackets <>.
Html elements could be semantics, meaning that they describe their meaning to both the browser and the developer. Examples are

tag that is for paragraph and del> tag that is for deleting text. They could also be non semantics, meaning that they say nothing about their contents and have no meaning example

. On the other hand, html elements could be classified based on whether they are container tags or empty tags.

HTML ELEMENTS

Html element is the name we use to describe everythin we can display on the web page. Elements tell the browser how to display the content. A typical html element consists of an opening tag, some attributes, the text content and a closing tag.
